About the Program

This application is a demonstration of the Django Web Framework for Python. The purpose of this program is to provide a solution to tracking finances. Tracking finances puts it in a bright light because it currently is not as functional as I would want it to be. Right now, the program puts registered purchases and incomes into a database and spits it back out nice and pretty for the user that is currently authenticated. Then, it also grabs these purchases and incomes and performs elementary statistics and spits the nice little boxes back out onto the statistics pages. I'm trying to get it to a better place and have more functionalities but I am slowly losing two things over the course of this project: ideas and motivation.
Beyond this, I would like to give credit where credit is due. The base of this project came from a tutorial series on Youtube. That being the Django Tutorials series from Corey Schafer. This tutorial helped me learn Django at a much deeper level than I thought I was capable of. These videos helped provided me with beautiful CSS and Bootstrap classes, as well as help with the creation of the forms. If I could talk to him, I would thank him because this series is so much better than the resources I was provided with to learn Django.
